# ServiceFlow Frontend

Frontend code lives here.

- `src/components` contains reusable UI and layout components.
- `src/features/*/components` contains domain-specific views and forms.
- `src/assets/styles/globals.css` contains global styling.
- `src/pages` is reserved for route-level view modules if pages are extracted from `src/app`.
- `public` contains static frontend assets.

`src/app` remains at the project root as the Next.js routing adapter.
